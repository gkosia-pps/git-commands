*** To check the global configuration user:

        git config --global --list

*** To check the local repo config:

        1. Navigate to the repo folder
        2. git config --list

*** To set/change the global configuration user:

        git config --global user.name "username"
        git config --global user.email "email"

*** To set the user for specific repo:

        1. Navigate to the repo folder
        2. git config user.name "username"
        3. git config user.email "email"
